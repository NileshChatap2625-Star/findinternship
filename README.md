# 🚀 FindInternship – AI Powered Internship Finder

> An intelligent AI-driven platform that helps students find relevant internships using resume analysis, ATS scoring, and personalized recommendations.

---

## 📌 Problem Statement

Students struggle to find relevant internships due to scattered platforms, lack of personalized recommendations, and inefficient search systems. This leads to wasted time, missed opportunities, and poor career decisions.

---

## 💡 Solution

FindInternship is a **centralized AI-powered web platform** that simplifies the internship search process by combining:

- 📄 Resume Analysis  
- 📊 Real-Time ATS Score  
- 🎯 Smart Internship Matching  

The system intelligently recommends internships based on user skills, resume content, and preferences, making the process **faster, smarter, and more effective**.

---

## ✨ Key Features

- 🔍 Smart Internship Search (filters: domain, location, stipend)  
- 🤖 AI-Based Personalized Recommendations  
- 📄 Resume Upload & Analysis  
- 📊 Real-Time ATS Score Generation  
- 🎯 Matched Internship Suggestions  
- 📌 Application Tracking Dashboard  
- 🧑‍💼 User Profile Management  
- 🛠️ Admin Panel (Approve/Reject Applications)  
- 📩 Offer Letter Workflow System  

---

## 🧠 AI Capabilities

- Resume parsing & keyword extraction  
- ATS score calculation  
- Intelligent internship matching  
- (Optional) Google Gemini API integration

       ↓  
Frontend (React - Lovable)  
       ↓  
Backend (Supabase)  
       ↓  
Database (PostgreSQL)  
       ↓  
AI Engine (ATS + Recommendation)  
      ↓  
Admin Panel → Offer System  
      ↓  
Cloud Deployment (Lovable & vercel)
